Plotting economic indicators with C3.js

For my first post I thought of doing a simple data transform and plot article. So let's just dive right in and use the awesome C3.js library to plot economic development indicators of a few economies including India and China. The data is taken from the Penn World Tables.

So first thing to do is to download the relevant csv from the source. For some really interesting economic datasets head on over to http://www.nber.org/data. There you'll find a link to the Penn-World Tables. We'll be looking at the growth accounting indicators.

Once downloaded we'll need to transform the csv data to JSON format so that it can be passed to C3. Python to the rescue!

Let's define a function that can take in a pandas dataframe and returns back the C3 friendly JSON containing the relevant data for an individual datapoint in the csv.

Next, we'll need to import the pandas library, read in the csv file and create a dictionary that maps the different datapoint names to their appropriate pandas dataframes. This is to iterate over them quickly while saving the individual json files.

Finally, we iterate over the dictionary items and save the json files.
